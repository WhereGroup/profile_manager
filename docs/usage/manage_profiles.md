# Managing QGIS profiles

![Profile Manager - Manage profiles](../static/profile_manager_ui_tab_manage-profiles.png)

## Creating a New Profile

1. Open Profile Manager
2. Click **Create Profile**
3. Enter a profile name
4. Confirm the creation

The profile becomes available the next time QGIS is started or when switching profiles.

---

## Copying a Profile

Copying allows you to duplicate a fully configured profile.

Steps:

1. Select the source profile
2. Click **Copy Profile**
3. Provide a new profile name
4. Confirm

This operation duplicates all settings and configuration data.

---

## Renaming a Profile

1. Select the profile
2. Click **Rename Profile**
3. Enter the new name
4. Confirm

---

## Deleting a Profile

1. Select the profile
2. Click **Delete Profile**
3. Confirm deletion

:::{caution}
Removing a profile deletes its configuration directory. Backups are strongly recommended before deletion.
:::
